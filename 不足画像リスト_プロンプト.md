## nanobananaai用プロンプト

### 1. フォレストハウスロゴ（logo.webp）
```
不動産会社「フォレストハウス」のロゴマークを作成してください。
- 会社名：フォレストハウス（FOREST HOUSE）
- 業種：マンション売却・不動産仲介
- イメージ：信頼感、自然、成長、プロフェッショナル
- 色：青系（#0066CCなど）を基調に、緑や自然を感じさせるアクセント
- サイズ：横長のロゴ（200x40px相当）
- スタイル：モダンで洗練されたデザイン、読みやすいフォント
- 要素：森や家のアイコンを組み合わせたデザインが望ましい
```

### 2. フォレストハウス白ロゴ（logo_white.webp）
```
上記のロゴを白一色のバージョンで作成してください。
- 背景が暗い色（フッター用）でも見やすい白いロゴ
- 同じデザインを白で表現
```

### 3. ファビコン（favicon.ico）
```
フォレストハウスのファビコンを作成してください。
- 32x32pxまたは16x16px
- ロゴの簡略版または森・家のアイコン
- 青系の色を使用
- シンプルで認識しやすいデザイン
```

### 4. OGP画像（ogp.webp）
```
フォレストハウスのOGP画像を作成してください。
- サイズ：1200x630px
- タイトル：「マンション売却なら成約率86.3%のフォレストハウス」
- サブタイトル：「奈良・大阪で購入希望者がお待ちです」
- デザイン：信頼感のある不動産会社のイメージ
- 色：青系を基調に、プロフェッショナルな印象
- ロゴと主要メッセージを含める
```

### 5. LINEアイコン（line_icon.webp）
```
LINE公式アカウントのアイコンを作成してください。
- サイズ：30x30px
- LINEの緑色（#00C300）を使用
- シンプルなLINEアイコンまたは「LINE」テキスト
```

### 6. LINE QRコード（line_qr.webp）
```
LINE公式アカウントのQRコード画像を作成してください。
- サイズ：150x150px
- 実際のQRコードまたはQRコード風のデザイン
- 中央にLINEロゴを配置
- 背景は白またはLINEの緑色
```

成約実績ページのヒーロー背景画像を作成してください。

## 画像作成の優先順位

1. **最優先**：ロゴ関連（logo.webp, logo_white.webp, favicon.ico）
2. **高優先**：OGP画像（ogp.webp）
3. **中優先**：LINE関連（line_icon.webp, line_qr.webp）
4. **中優先**：成約実績ヒーロー（results_hero.webp）
5. **低優先**：お客様の声画像（results_voice1-6.webp）

---

## 注意事項

- すべての画像はWebP形式で作成してください
- ファビコンのみ.ico形式が必要です
- サイズは推奨サイズを目安にしてください
- 色はフォレストハウスのブランドカラー（青系：#0066CCなど）を意識してください
- 人物画像は実在の人物に似せないよう注意してください（イラストまたはAI生成推奨）
